# MNHAHub v6.6

Script desenvolvido para os jogadores de roblox que usam executores conseguirem satisfazer seus desejos de hackear um jogo no roblox, na minha opinião, quem faz isso é muito iteligente.

## 🚀 Funcionalidades
- Noclip
- Fly
- Infinite Jump
- Aimbot
- Wall check
- Team check
- Alive check
- FPS Unlocker
- etc...
